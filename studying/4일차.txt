ⓐfor문에 대하여 (반복문)
	for문 표현식
		for (초기화 ; 조건식 ; 증감식) {
			조건식이 참일 때 수행될 문장들을 적는다
		}

	누적 하는 방법
		1~n까지의 합
			int sum = 0;
			for (int i=1 ; i<=n ; i++) {
			
			//결과에 대한 자신 = 결과에 대한 자신 + 대상
			sum = sum + 1
			}
	★for 안에서 선언하였거나 {} 안에서 선언한 경우 그 {} 가 끝나면 효율을 잃음
		ex) for (int i = 1 ; 1 < 5 ; i++) { }
			System.out.print(i); =====> 오류임 for 안에서 i를 선언하였기 때문에 밖에서 사용 불가능

	i는 왠만하면 for문에 귀속시켜야 한다.
